# 🚀 Offinity AI

> Build complete full-stack apps from simple English prompts — no coding required, fully offline, with visual editing.

---

## 🧠 Overview

**Offinity AI** is an offline-first AI-powered development platform that enables users to build complete software applications using natural language.

Unlike traditional AI coding tools, Offinity AI allows users to generate, edit, debug, and test full-stack applications using both **offline local models** and **optional online APIs** — all within a unified environment.

---

## 💡 Key Features

### 🔹 Full-Stack App Generation

* Generate complete applications from a single prompt
* Supports frontend, backend, and database integration
* Multi-language support (Python, JavaScript, Java, Rust, etc.)

---

### 🔹 Offline + Online AI Support

* Fully functional with local models (LM Studio, Ollama)
* Optional cloud API integration (OpenAI, Anthropic, OpenRouter)
* No mandatory subscription required

---

### 🔹 Beginner-Friendly (No Coding Required)

* Build apps using simple English prompts
* No prior programming knowledge needed

---

### 🔹 Visual Builder

* Drag, resize, and edit UI elements
* Modify design visually without touching code
* AI-assisted UI improvements

---

### 🔹 API Flow Visualization

* Visual representation of pages, APIs, and databases
* Connect components using arrows
* Automatically updates real code

---

### 🔹 Smart Code Editing

* Modify files using natural language
* Patch-based updates for efficiency and stability
* Maintains consistency across large projects

---

### 🔹 Debugging & Auto-Fix

* Detects errors across the project
* Suggests and applies fixes using AI

---

### 🔹 Test Automation

* Runs tests automatically
* Fixes failing tests using AI
* Supports multiple frameworks

---

### 🔹 Git Integration

* Automatic version control
* Undo changes anytime
* View differences like GitHub

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/offinity-ai.git
cd offinity-ai
```

### 2. Run Project

```bash
python main.py
```

---

### 🔹 Demo Mode (No Setup Required)

* Runs with pre-configured responses
* Allows quick testing without local models

---

### 🔹 Full Mode (Optional)

1. Install LM Studio or Ollama
2. Load a local model
3. Connect to Offinity AI
